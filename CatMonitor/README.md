# 🐱 Cat Monitor

Transmite audio en vivo de un teléfono Android a otro vía WiFi.
Perfecto para escuchar a tu gato desde otra habitación. 😺

---

## 🚀 Cómo obtener el APK (sin instalar nada)

### Paso 1 — Sube el proyecto a GitHub

1. Entra a **https://github.com** y crea una cuenta gratis si no tienes
2. Haz clic en **"New repository"** (botón verde)
3. Ponle nombre: `cat-monitor`, déjalo en **Public** → **"Create repository"**
4. En la siguiente pantalla haz clic en **"uploading an existing file"**
5. Descomprime el ZIP y sube **todos los archivos y carpetas**
6. Haz clic en **"Commit changes"**

### Paso 2 — GitHub compila el APK solo 🤖

1. Ve a la pestaña **"Actions"** de tu repositorio
2. Verás **"Build Cat Monitor APK"** ejecutándose 🟡
3. Espera ~3-5 minutos hasta que aparezca en verde ✅

### Paso 3 — Descarga el APK

1. Haz clic en el workflow completado ✅
2. Baja hasta **"Artifacts"**
3. Descarga **"CatMonitor-APK"**
4. Dentro hay el archivo `app-debug.apk` 🎉

### Paso 4 — Instala en ambos teléfonos

- Pasa el APK por WhatsApp, Google Drive o cable USB
- Abre el `.apk` → acepta "instalar de fuentes desconocidas"
- ¡Instala en ambos teléfonos!

---

## 📱 Cómo usar

**Teléfono viejo (junto al gato):**
1. Abre Cat Monitor → 📡 **MODO GATO** → **Iniciar**
2. Anota la **IP** que aparece

**Tu teléfono:**
1. Abre Cat Monitor → 🎧 **MODO ESCUCHA**
2. Escribe la **IP** del otro teléfono → **Conectar**
3. ¡A espiar al minino! 🐾

> ⚠️ Ambos teléfonos deben estar en el **mismo WiFi**
